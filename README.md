# 📊 Simply Supported Beam Visualizer (Python GUI)

A Python-based GUI application to analyze and visualize the structural behavior of a simply supported beam under a point load. The tool automatically generates Shear Force Diagrams (SFD) and Bending Moment Diagrams (BMD) with an intuitive and modern interface.

# 🚀 Features

**📐 Instant Visualization**
Enter beam length, load magnitude, and load position to generate SFD and BMD instantly.


**🎨 Modern GUI**
Built using CustomTkinter for a clean and user-friendly interface.


**✅ Input Validation**
Ensures valid inputs for beam length, load, and position to prevent incorrect calculations.


**📍 Highlighted Critical Points**
Clearly marks load position and key changes in shear force and bending moment.


**💾 Export Diagrams**
Save high-quality diagrams as PNG files with annotated beam details.


**📂 Custom Save Options**
Choose file name and location before saving.


**🛠️ Tech Stack**
Python 3.11+
Matplotlib – for plotting diagrams
NumPy – for numerical computations
Tkinter / CustomTkinter – for GUI development
